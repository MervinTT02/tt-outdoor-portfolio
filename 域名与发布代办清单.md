## A. 你需要完成（必须本人账号操作）

### 1) 在阿里云购买域名
- 进入阿里云域名注册页，先查 `ttontheway.com`。
- 若不可用，购买 `ttmervin.com`。
- 完成实名模板与支付。

### 2) 把域名 DNS 托管切到 Cloudflare（推荐）
- 注册/登录 Cloudflare 免费版。
- `Add a domain`，输入你买到的域名。
- Cloudflare 会给你 2 条 NS 服务器地址（示例：`xxx.ns.cloudflare.com`）。
- 回到阿里云域名控制台，把该域名 DNS 服务器改为 Cloudflare 提供的 2 条 NS。
- 等待生效（通常几分钟到 24 小时）。

## B. 我可以继续代办（技术部分）

### 1) 推送网站到 GitHub 仓库
建议仓库名：`tt-outdoor-portfolio`

本地命令（在项目根目录执行）：

```bash
git add .
git commit -m "feat: launch TT outdoor photography portfolio"
git branch -M main
git remote add origin git@github.com:MervinTT02/tt-outdoor-portfolio.git
git push -u origin main
```

### 2) 连接 Cloudflare Pages 自动部署
- Cloudflare -> `Workers & Pages` -> `Create application` -> `Pages` -> `Connect to Git`
- 选仓库：`MervinTT02/tt-outdoor-portfolio`
- 构建配置：
  - Framework preset: `None`
  - Build command: 留空
  - Build output directory: `/`
- 点击 `Save and Deploy`

### 3) 绑定正式域名
- Pages 项目 -> `Custom domains` -> `Set up a custom domain`
- 添加：
  - `ttontheway.com`（或备选域名）
  - `www.ttontheway.com`
- 等待 SSL 证书自动签发完成（状态变为 Active）。

## D. 备注
- 你的网站托管在 Cloudflare 全球节点，不是中国大陆机房，一般不要求 ICP 备案。
